5. 주요 사용법

5.1 센스리더 오류대응

5.1.1 label, input(checkbox,radio) 마크업 : 레이블 불인식

(마크업 오류 방지) id, for 제공없이 label > input 형태의 마크업 시 레이블 불인식됨. 어떤 형태이든 id, for 맞춰서 제공

5.1.2 버튼 누르기 동작 시 enter 키 사용불가

(누르기 동작 지원) 센스리더 동작 중 space키와 enter키 동시 사용 가능해야 함.
onclick, keyup 이벤트만 사용. keydown, keypress 이벤트 안됨.
